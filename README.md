This is fat tools for my projects.

It need to manual merge with custom projects.

## Included

* Build js (react) with webpack
* Build less, jade(pug)
* Dev server browser sync + webpack
* Use pouchdb(couchdb) for store redux data. Offline support

## Get started
```
npm install
gulp help
```

## Requirements:
 - node v6
